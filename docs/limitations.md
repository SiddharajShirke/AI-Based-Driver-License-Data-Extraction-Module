# Limitations

This document describes the current limitations of the system.

---

## 1. Document Type Support
- Only supports driver's licenses (no passports, ID cards, or other documents)
- Optimized for major countries (US, India, UK, Australia, Canada)

## 2. Batch Processing
- Single image at a time only
- No bulk upload or simultaneous processing
- 3-8 second response time per image

## 3. Internet Dependency
- Requires active internet connection for Gemini API
- Subject to API rate limits and costse

## 4. Data Storage
- No persistence or history
- Results not saved automatically
- No database integration

## 5. Security and Authentication
- No user authentication
- Data sent to external API (Google Gemini)
- No rate limiting or access controls
